Class SmartsheetImpl

This is the implementation of Smartsheet interface.

Thread Safety: This class is thread safe because all its mutable fields are safe-guarded using AtomicReference to ensure atomic modifications, and also the underlying HttpClient and JsonSerializer interfaces are thread safe.
