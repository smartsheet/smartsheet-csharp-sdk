Class Row.AddRowBuilder

A convenience class for creating a Row with the necessary fields for inserting into a list of Rows.

Inheritance
System.Object
Row.AddRowBuilder
Namespace: Smartsheet.Api.Models
Assembly: smartsheet-csharp-sdk.dll
Syntax
public class AddRowBuilder : object
Remarks

If multiple rows are specified in the request, all rows must be inserted at the same location (i.e. the toTop, toBottom, parentId, siblingId, and above attributes must be the same for all rows in the request).

Declaration
public AddRowBuilder(bool? toTop, bool? toBottom, long? parentId, long? siblingId, bool? above)
Parameters
Type Name Description
System.Nullable<System.Boolean> toTop

Indicates that the row should be added to the top of the sheet.
System.Nullable<System.Boolean> toBottom

Indicates that the row should be added to the bottom of the sheet, or, if used in conjunction with parentId, added as the last child of the parent.
System.Nullable<System.Int64> parentId

Indicates that the row should be added as the first child of this row.
System.Nullable<System.Int64> siblingId

Indicates that the row should be added as the next row at the same hierarchical level of this row.
System.Nullable<System.Boolean> above

Optionally used in conjunction with siblingId with a value of true to indicate that the row should be added above the specified sibling row.
Remarks

If multiple rows are specified in the request, all rows must be inserted at the same location (i.e. the toTop, toBottom, parentId, siblingId, and above attributes must be the same for all rows in the request).
